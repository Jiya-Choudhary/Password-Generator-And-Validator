# 🔐 Password Generator & Validator

A Python-based Password Generator and Validator that generates secure passwords and validates password strength using custom exception handling.

## ✨ Features

- 🔑 Generate Random Passwords
- ✅ Password Strength Validation
- 🔢 Numeric Character Check
- ⚠️ Custom Exception Handling
- 🛡️ Basic Password Security Rules

---

## 🛠️ Tech Stack

- Python 3
- Random Module
- String Module
- Exception Handling
